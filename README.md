一个基于 LangChain 框架构建的 AI 智能体项目
## 项目概述

MyAgent-1 是一个演示如何使用 LangChain 框架构建智能 AI 助手的项目。它展示了如何集成不同的 LLM API、创建自定义工具、实现动态提示以及构建可交互的智能体。

## 主要特性

- **多模型支持**: 支持 DeepSeek API 和阿里云百炼 API（通义千问）
- **智能体框架**: 基于 LangChain 的智能体架构
- **自定义工具**: 内置多种实用工具函数
- **动态提示**: 根据用户角色调整回答详细程度
- **流式响应**: 支持思考过程的实时显示
- **环境配置**: 通过环境变量管理 API 密钥
